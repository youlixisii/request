# 更新日志

## [1.0.0] - 2024-01-01

### 新增功能

#### 核心层 (request-core)
- ✅ 实现了 `Requestor` 接口定义
- ✅ 实现了依赖注入机制（DIP原则）
- ✅ 实现了请求缓存功能
  - 支持内存缓存
  - 支持持久化缓存（localStorage）
  - 支持自定义缓存键
  - 支持自定义缓存有效期
  - 支持自定义缓存有效性检查
- ✅ 实现了请求重试功能
  - 支持自定义重试次数
  - 支持自定义重试延迟
  - 支持自定义重试条件
- ✅ 实现了并发控制功能
  - 支持限制最大并发数
  - 自动队列管理
- ✅ 实现了请求幂等功能
  - 基于请求内容哈希
  - 防止重复提交
  - 支持自定义幂等键生成

#### 实现层 (request-implementations)
- ✅ 实现了基于 Axios 的请求器
  - 完整的 HTTP 方法支持（GET, POST, PUT, DELETE, PATCH）
  - 事件系统支持
  - 请求/响应拦截器
  - 错误处理

#### 业务层 (request-bus)
- ✅ 实现了业务请求器初始化
- ✅ 实现了统一的响应格式处理
- ✅ 实现了请求拦截器
  - 自动添加 Token
  - 自动添加时间戳
- ✅ 实现了响应拦截器
  - 统一错误处理
  - 自动处理认证失败
- ✅ 实现了便捷的 API 方法
  - `get()`, `post()`, `put()`, `del()`, `patch()`
  - `request()` 通用方法

### 文档
